# API Contracts: Sirva CMS Marketplace Platform

**Version**: v1  
**Base path**: `/v1` (e.g. `https://api.example.com/v1`)  
**Auth**: Bearer token for admin/vendor; API key or token for consuming Sirva applications (widget delivery).

All request/response bodies are JSON. Timestamps are ISO 8601. Pagination uses `page` (1-based) and `pageSize` (default 20, max 100).

See individual files for each endpoint family.
